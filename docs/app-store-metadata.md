# App Store Metadata Draft

## App Name

Charm ID

## Subtitle

Camera identification for small items

## Category

Productivity or Business

## Short Description

Register small items from multiple angles, identify them with your iPhone camera, and improve matching with confirmed results.

## Longer Description Draft

Charm ID helps teams and makers manage charms, accessories, parts, and other small items that are hard to distinguish at a glance.

Register each item with a management number and six guided camera angles: front, back, right side, left side, top side, and bottom side. When you need to identify an item, capture one or more angles and review the best matching candidates.

Every confirmed identification can be added back into the item's learning data, so the reference library becomes stronger as it is used in real workflows.

## Keywords Draft

inventory, camera, identification, catalog, parts, accessories, item management, visual search

## Review Notes Draft

The app can be used without login in the initial version. Data is stored locally on the device unless cloud sync is added in a later release.

The camera is used to capture item photos for registration and identification.

Use "デモデータで試す" on the first screen to load sample models with all six capture angles.
