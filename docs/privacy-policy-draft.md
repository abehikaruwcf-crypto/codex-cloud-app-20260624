# Privacy Policy Draft

Last updated: 2026-06-25

Charm ID helps users register and identify small items using camera photos.

## Data Collected

The app may store:

- Item names or management numbers entered by the user.
- Photos captured or selected by the user for item registration and identification.
- Matching results and user-confirmed corrections.

## Local Storage

The current prototype stores data locally on the user's device. The data is not transmitted to a remote server by the prototype.

## Camera and Photo Library

The app requests camera access to capture item photos. It may request photo library access when the user chooses to select or save images.

## Future Cloud Sync

If cloud sync is added, this policy must be updated before release to describe:

- What data is uploaded.
- Where it is stored.
- How users can delete their data.
- Whether data is used to improve matching models.

## Contact

Contact information must be added before App Store submission.
